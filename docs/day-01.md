# Day 1 Notes

## Today's Goal

Create the base project structure and initialize Git.

## Concepts

- `frontend`: the Next.js application that users see in the browser.
- `backend`: the FastAPI service that exposes APIs and later talks to files,
  databases, and AI models.
- `docs`: learning notes and daily progress records.
- `.gitignore`: tells Git which generated or private files should not be
  committed.
- `README.md`: explains what the project is and how to run it.

## Git Habit

At the end of each day:

1. Check changes with `git status`.
2. Review changed files.
3. Commit meaningful work.
4. Push to GitHub.
